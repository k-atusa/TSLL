def 소스 s를 컴파일:
  s를 토큰화;
  s를 순회하며 모듈 임포트를 찾음:
    이미 컴파일된 모듈이면 네임스페이스 가져오기/아니면 해당 모듈부터 컴파일;
  타입 호이스팅 대상인 구조체, 열거형을 찾음:
    순회를 반복하며 타입과 메모리 크기를 결정:
      순회 후 타입 정보가 변경된 것이 없는데 메모리 크기가 미확정인 것이 있다면 순환 참조이므로 오류;
  제어 호이스팅 대상인 전역변수, 함수를 찾음:
    네임스페이스에 전역변수를 등록;
    네임스페이스에 함수를 등록;
  s에 대한 선행 컴파일을 끝내고 본 컴파일 시작:
    함수 단위로 컴파일;
  AST를 컴파일 완료 모듈에 등록;

def 위치노드 l:
  int source_id;
  int source_line;

def 값노드 v;
  int value_type;
  int value_int;
  float value_float;

def 토큰 t:
  int token_type;
  l location;
  v value;
  string text;

def 파싱노드 a:
  int ast_type;
  l location;
  v value;
  a* type_node;
  a* child_node;
  int child_len;
